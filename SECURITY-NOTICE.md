# Security Notice

This repository contains a reference specification and early-stage scaffolding
for the XRP Safe Transfer & Recovery Protocol (XSTRP).

Status:
- RFC-XSTRP-0001 is frozen as Draft v1.0
- Reference architecture is complete
- No external peer review has been conducted yet

Important Notice:
This specification has not yet undergone external peer review.
All security claims, guarantees, and threat mitigations described herein
are subject to future independent audit and review.

Nothing in this repository should be treated as production-ready code.
